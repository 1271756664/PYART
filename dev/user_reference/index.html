User Reference Manual¶

Release

1.10

Date

April 24, 2019

This is the Py-ART user community’s reference guide, and it covers the most commonly used public functions, modules and classes. If more detail is required, the Developer Reference Manual is an exhaustive listing of all private and public functions, modules and classes.

  • Core (pyart.core)
  • Input and output (pyart.io)
  • Auxiliary input and output (pyart.aux_io)
  • Radar Corrections (pyart.correct)
  • Radar Retrievals (pyart.retrieve)
  • Graphing (pyart.graph)
  • Mapping (pyart.map)
  • Filters (pyart.filters)
  • Utilities (pyart.util)
  • Bridging to other toolkits (pyart.bridge)
  • Testing Utilities (pyart.testing)

Packages level functions:

load_config([filename])

Load a Py-ART configuration from a config file.

test([verbose])

Invoke the Py-ART test suite.
