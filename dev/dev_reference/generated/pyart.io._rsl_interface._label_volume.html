pyart.io._rsl_interface._label_volume¶

pyart.io._rsl_interface._label_volume(volume, radar)¶

Add labels for dealiasing to a _RslVolume object from a radar object.

This function does not set all parameter in the _RslVolume suitable for writing out the volume, rather it set those parameters which must be set prior to using pyart.correct._fourdd_interface.fourdd_dealias.

Parameters:

volume : _RslVolume

Volume object to which parameters will be set as needed prior to dealiasing. Object is manipulated in-place.

radar : Radar

Radar object from which parameters are taken.
