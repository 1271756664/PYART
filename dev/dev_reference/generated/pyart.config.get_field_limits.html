pyart.config.get_field_limits¶

pyart.config.get_field_limits(field, container=None, selection=0)[source]¶

Return the data limits from the configuration file for a given field, radar and sweep

Parameters:

field: str :

Field name.

container: Radar, Grid or None :

This is an optional parameter that will be use to get informations related to the field, like for instace nyquist velocity.

selection: int :

selection of the data in the container, case container is a Radar this is the sweep to be considered
Returns:

vmin, vmax: 2-tuplet of float :

Minimun and Maximun teorical value for field, if field is not in the configuration file returns (None, None).
