pyart.io.nexrad_level2.NEXRADLevel2File.get_range¶

NEXRADLevel2File.get_range(scan_num, moment)[source]¶

Return an array of gate ranges for a given scan and moment.

Parameters:

scan_num : int

Scan number (0 based).

moment : ‘REF’, ‘VEL’, ‘SW’, ‘ZDR’, ‘PHI’, or ‘RHO’

Moment of interest.
Returns:

range : ndarray

Range in meters from the antenna to the center of gate (bin).
