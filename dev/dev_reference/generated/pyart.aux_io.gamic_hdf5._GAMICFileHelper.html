pyart.aux_io.gamic_hdf5._GAMICFileHelper¶

class pyart.aux_io.gamic_hdf5._GAMICFileHelper(hfile)[source]¶

A class to help read GAMIC files.

This class has methods which performs looping routines which are useful when reading GAMIC files.

Methods
