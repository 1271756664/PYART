pyart.io._rsl_interface._RslSweep¶

class pyart.io._rsl_interface._RslSweep¶

A object for accessing RSL Sweep data and header information.

This class should not be initalized from within Python. _RslSweep objects are returned from the _RslVolume.get_sweep method.

Attributes

sweep_num

(int) Interger sweep number.

elev

(float) Mean elevation angle for thr sweep. -999.0 for RHI sweeps.

azimuth

(float) Azumuth for the sweep. -999.0 for PPI scans.

beam_width

(float) Beam width in degrees. Can also be found in _RslRay objects.

vert_half_bw

(float) Vertical beam width divided by 2.

horz_half_bw

(float) Horizontal beam width divided by 2.

nrays

(int) Number of rays in the sweep.

Methods

get_data()

Return the two-dimensional data contained in the sweep.

get_ray(ray_number)

Return a _RslRay for a given ray.
