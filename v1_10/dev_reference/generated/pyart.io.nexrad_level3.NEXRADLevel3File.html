pyart.io.nexrad_level3.NEXRADLevel3File¶

class pyart.io.nexrad_level3.NEXRADLevel3File(filename)[source]¶

A Class for accessing data in NEXRAD Level III (3) files.

Attributes

text_header

(dic) File textual header.

msg_header

(dic) Message header.

prod_descr

(dic) Product description.

symbology_header

(dict) Symbology header.

packet_header

(dict) Radial data array packet header.

radial_headers

(list of dicts) List of radials headers

raw_data

(array) Raw unscaled, unmasked data.

data

(array) Scaled, masked radial data.

_fh

(file-like) File like object from which data is read.

Return the latitude, longitude and height of the radar.

get_range()

Return an array of gate range spacing in meters.

get_volume_start_datetime()

Return a datetime of the start of the radar volume.
